## How to Create Echoes & Stories (Raji way)
- Echoes are **your original fictional heroes** inspired by epic virtues (dharma as contextual duty/virtue like Rama's honor or universal kindness; karma as positive actions leading to growth).
- Stories: Short, personal, positive retellings or new tales. Focus on dharma choices (right action in context), pure intention, family/clan uplift, courage.
- **We do not pre-fill stories.** The core is UGC — players write their own. We only provide beautiful prompts and examples grounded in positive lessons from epics (Hindu svadharma + Buddhist intention focus).
- Example prompt in app: "When I chose family duty like Arjuna learned (detached righteous action)..." → Make it yours. Positive karma return through kindness/sharing.
